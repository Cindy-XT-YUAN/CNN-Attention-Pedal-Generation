# Demo

This folder contains example MIDI files demonstrating the effect of automatic pedal generation.

Each example includes:

- `<name>.mid`: Original MIDI without sustain pedal (CC64)
- `<name>_withPed.mid`: MIDI with generated pedal
- `<name>_withPed.png`: Visualization of pedal signals

These examples illustrate how the model improves musical expressiveness by adding realistic sustain pedal control.
